Installation:

1. sudo apt install software-properties-common
2. sudo add-apt-repository ppa:deadsnakes/ppa
3. sudo apt update
4. sudo apt dist-upgrade
5. sudo apt install python3.13 
6. install gcc (sudo apt install gcc)
7. install cc (sudo apt install cc)
8. sudo apt install build-essential
9. sudo apt install pkg-config
10. python3.13 -m venv venv
11. source venv/bin/activate
12. upgrade tools (pip install --upgrade pip setuptools wheel)
13. python update to 3.13 (new raspberri pi OS comes with 3.11, maybe will work):
  A. sudo apt install build-essential zlib1g-dev libncurses5-dev libgdbm-dev libnss3-dev libssl-dev libreadline-dev libffi-dev libsqlite3-dev wget libbz2-dev
  B. installed latest python with: https://computingforgeeks.com/how-to-install-python-on-debian-linux/
14. install dependecies (python install_dependencies.py)
